Operators and Loops

Comparison Operators

Can be a strict comparison === or non-strict ==. You can also compare greater than, less than, not equal to, etc.

  • Returns a boolean
  • === Will only return true if it's the same data type and has the same value.
  • == Will return true even if you compare a number 4 and a string `4`
  • ! Means not equal to

Assignment Operator

A single = assigns a value to whatever is to the left of it.

Loops

Loops offer a way to programatically repeat steps. The 2 types of loops this we'll focus on are if and while statements.

  • A for loop will repeat till the condtion returns false.
  • A while loop executes only while the condition is true
