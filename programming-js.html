Control Flow

Control Flow is the order in which the computer executes code

The computer executes the code from top to bottom, calling functions when it reachs lines invoking them.

Use an if else statement to test if certains conditions exist or not. Such as if a form to register a new account is filled out completely. Control Flow means being mindful of our programs structure and how it affects the order of execution.

JS Functions

A JavaScript function is a block of code designed to perform a particular task. It'll only executed when invoked. To declare a function, use the function keyword followed by a name, parentheses, and curly braces where the code to be executed resides. The parentheses can take arguments or parameters. A function can be invoked by calling it, an event, or on load up of the page. If you want to use the value the function returns, you need a return statement. To call the function type its name followed by parentheses. Variables declared from within a function are local to that function. Local variables are only accessible from within their function. Local variables are created when a function starts, and deleted when the function is completed.
